# 콜백 SMS 앱 설치 가이드

개발 경험이 없어도 이 가이드를 따라하면 앱을 설치할 수 있습니다.

---

## 1단계 — Android Studio 설치

1. [https://developer.android.com/studio](https://developer.android.com/studio) 접속
2. "Download Android Studio" 클릭 → 설치 파일 실행
3. 설치 마법사에서 모두 "Next" → "Finish"
4. 처음 실행 시 "Standard" 설정으로 진행 (SDK 자동 다운로드, 5~10분 소요)

---

## 2단계 — 프로젝트 열기

1. Android Studio 실행
2. 상단 메뉴 **File → Open...**
3. `새 폴더 (2)/CallbackSMS` 폴더 선택 → **OK**
4. 우측 하단에 "Gradle sync" 진행 중 표시 → 완료될 때까지 대기 (2~5분)
   - 인터넷 연결 필요 (라이브러리 자동 다운로드)

---

## 3단계 — 핸드폰 연결

### USB 케이블로 연결하는 방법
1. 핸드폰 **설정 → 휴대전화 정보 → 소프트웨어 정보**
2. **빌드 번호**를 7번 연속 탭 → "개발자 모드 활성화" 메시지 확인
3. **설정 → 개발자 옵션 → USB 디버깅** 켜기
4. USB로 PC에 연결
5. 핸드폰에 "USB 디버깅 허용?" 팝업 → **허용**

### Android Studio에서 확인
- 상단 툴바에 내 핸드폰 모델명이 표시되면 연결 완료

---

## 4단계 — 앱 빌드 & 설치

1. Android Studio 상단 툴바에서 **▶ (Run)** 버튼 클릭
   - 또는 단축키: `Shift + F10`
2. 처음에는 빌드에 3~5분 소요
3. 핸드폰에 앱이 자동 설치되어 실행됩니다

---

## 5단계 — 권한 허용

앱이 실행되면:
1. 화면에 **권한 허용** 안내가 표시됩니다
2. 큰 버튼을 탭하면 권한 요청 팝업이 뜹니다
3. 아래 권한을 **모두 허용**해야 합니다:
   - 전화 상태 읽기
   - 통화 기록 읽기  
   - 문자 메시지 전송
   - 연락처 읽기
   - 알림 표시

> 삼성 갤럭시의 경우 배터리 최적화에서 콜백SMS 앱을 "배터리 사용 제한 없음"으로 설정하면 더 안정적으로 동작합니다.
> 설정 → 배터리 → 배터리 사용 최적화 → 모든 앱 → 콜백SMS → 최적화 안함

## 앱 사용법

### 기본 사용
1. **홈 화면** 큰 원형 버튼을 탭 → **ON** 상태로 변경
2. 누군가에게 전화를 걸고 끊으면 → 자동으로 문자 전송

### 메시지 템플릿 관리 (템플릿 탭)
- 기본 템플릿 4개가 미리 준비되어 있습니다
- **+** 버튼으로 새 템플릿 추가
- 원하는 템플릿에서 **"선택"** 버튼 탭 → 해당 메시지로 전송

### 변수 사용
템플릿 내용에 아래 변수를 넣으면 자동으로 바뀝니다:
- `{이름}` → 전화한 상대방 연락처 이름
- `{시간}` → 전송 시각 (예: 14:30)
- `{날짜}` → 전송 날짜 (예: 5월 1일)

### 예시 템플릿
```
{이름}님, 방금 전화드렸는데 연결이 안 됐네요.
{시간}에 다시 연락드릴게요!
```

### 설정 옵션 (설정 탭)
| 설정 | 설명 |
|------|------|
| 발신 통화 종료 후 | 내가 건 전화 종료 시 문자 전송 |
| 부재중 수신 후 | 상대방이 못 받은 전화에 자동 답장 |
| 최소 통화 시간 | N초 미만이면 전송 안 함 (실수 방지) |
| 활성 시간대 | 특정 시간대에만 작동 |

## 자주 묻는 문제

**Q: 앱이 꺼져 있어도 문자가 가나요?**
→ ON 상태이면 백그라운드 서비스가 계속 동작합니다. 핸드폰을 재시작해도 자동으로 다시 켜집니다.

**Q: 특정 사람에게는 안 보내고 싶어요**
→ 현재 버전에서는 전체 발송이 기본입니다. 특정 번호를 제외하려면 앱을 잠시 OFF 후 전화하세요.

**Q: 앱 빌드할 때 오류가 나요**
→ Android Studio 상단 "Build → Clean Project" 후 다시 빌드 시도.
→ 또는 인터넷 연결 확인 후 "File → Sync Project with Gradle Files" 클릭.

## 앱 구조 (참고용)

```
홈         — ON/OFF 토글, 오늘/전체 전송 통계, 현재 템플릿 확인
템플릿     — 메시지 추가/수정/삭제, 활성 템플릿 선택
기록       — 전송된 문자 내역, 성공/실패 여부
설정       — 트리거 설정, 활성 시간대, 최소 통화 시간
```
