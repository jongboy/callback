# 쉬운 설치 방법 (Android Studio 불필요)

GitHub 무료 계정만 있으면 클라우드에서 자동으로 APK를 만들어줍니다.

---

## 준비물
- PC (Windows/Mac 모두 가능)
- GitHub 계정 (없으면 무료로 가입)
- Android 핸드폰

---

## 1단계 — GitHub 계정 만들기

1. [https://github.com](https://github.com) 접속
2. "Sign up" → 이메일/비밀번호 입력 → 가입 완료

이미 계정이 있으면 1단계 건너뜀

---

## 2단계 — 새 저장소 만들기

1. 로그인 후 우측 상단 **+** 버튼 → **New repository**
2. Repository name: `CallbackSMS` 입력
3. **Private** 선택 (코드 비공개)
4. **Create repository** 클릭

---

## 3단계 — 파일 업로드

1. 저장소 페이지에서 **"uploading an existing file"** 링크 클릭
2. `새 폴더 (2)/CallbackSMS` 폴더를 **통째로 드래그&드롭**
   - 폴더 안의 모든 파일과 폴더가 업로드됩니다
3. 페이지 아래 **Commit changes** 클릭

> 파일이 많아서 드래그&드롭이 안 되면 아래 GitHub Desktop 방법 사용

---

## 4단계 — 빌드 기다리기

1. 저장소 상단 탭에서 **Actions** 클릭
2. "APK 빌드" 워크플로우가 자동으로 시작됩니다
3. 초록 체크(✓) 표시될 때까지 기다리기 (약 5~10분)

---

## 5단계 — APK 다운로드

1. Actions 탭 → 완료된 빌드 클릭
2. 페이지 맨 아래 **Artifacts** 섹션
3. **"콜백SMS-설치파일"** 클릭 → ZIP 파일 다운로드
4. ZIP 압축 풀면 `app-debug.apk` 파일이 있습니다

---

## 6단계 — 핸드폰에 설치

1. **APK 파일을 핸드폰으로 전송**
   - USB로 연결 후 파일 복사, 또는
   - 카카오톡/메시지로 파일 전송, 또는
   - 구글 드라이브 등 클라우드에 올려서 다운로드

2. **알 수 없는 앱 설치 허용**
   - 설정 → 보안 → **알 수 없는 앱 설치** 허용
   - (Samsung 기준: 설정 → 생체인식 및 보안 → 출처를 알 수 없는 앱 설치)

3. APK 파일을 탭 → **설치** 클릭

---

## 7단계 — 앱 실행 및 권한 허용

1. 앱 실행
2. 큰 버튼 탭 → 권한 요청 팝업에서 **모두 허용**
3. 버튼이 **ON** 상태가 되면 완료!

---

## 파일 업로드가 잘 안 될 때 — GitHub Desktop 사용

1. [https://desktop.github.com](https://desktop.github.com) 에서 GitHub Desktop 설치
2. 로그인 → Clone Repository → CallbackSMS 선택
3. `CallbackSMS` 폴더 내용을 복사해서 붙여넣기
4. GitHub Desktop에서 "Commit to main" → "Push origin" 클릭

---

## 요약

```
GitHub 가입 → 저장소 생성 → 파일 업로드
→ Actions에서 자동 빌드 (5~10분)
→ APK 다운로드 → 핸드폰에 설치
```

이 방법은 Android Studio 설치 없이, 코드를 건드리지 않고 APK를 받을 수 있습니다.
